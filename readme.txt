Programmet på Heroku kører, så der kan testes igennem den.

Hvis du vil starte din egen app skal følgende ske

1. I loginBruger, skal stierne ændres fra APP til folder
2. Du skal i test oprette 52 uger, så links kan blive oprettet
3. Du skal oprette en testbruger med ADmin login fra test
4. Det burde fungerer herefter


Testfirst fungerer og kan ses alle funktionelle i bilag i Rapporten.
